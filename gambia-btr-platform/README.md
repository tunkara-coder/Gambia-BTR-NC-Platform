# The Gambia BTR/NC Digital Transparency Platform

A web-based platform to streamline The Gambia's Biennial Transparency Report and National Communication reporting under the UNFCCC.

## Setup Instructions

1. **Download Files**: Save all files in the same folder structure
2. **Open in VSCode**: 
   - Open VSCode
   - File > Open Folder > Select the project folder
3. **Run the Application**:
   - Right-click on `index.html`
   - Select "Open with Live Server" (install Live Server extension if needed)
   - Or simply open `index.html` in your web browser

## File Structure

- `index.html` - Main application interface
- `styles/main.css` - Styling and layout
- `scripts/app.js` - Application logic and functionality
- `data/gambia-config.json` - Gambia-specific configuration
- `README.md` - This file
